oBonoB: Version Alpha

oBonoB est actuellement en version Alpha ce qui signifie que certaines pages sont manquantes et que certaines fonctionnalités peuvent être incomplètes ou présenter des bugs.

oBonoB n'est pas encore disponible publiquement mais cette première version Alpha permet de tester les fonctionnalités de la plateforme plus en profondeur qu'un simple déploiement en local.
Ainsi, nous comptons sur toi pour signaler quelconque bug ou disfonctionnement (et il y en aura sûrement :P) à l'adresse temporaire suivante: obonob.app[at]gmail.com


Les technos utilisées

Pour ceux éventuellement intéressés par la façon dont a été codée la plateforme, voici les technologies exploitées par le site.

Backend

Le backend est propulsé par les outils suivants:

  • NodeJS comme langage backend
  • ExpressJS pour la gestion des routes
  • MongoDB pour la base de données
  • Socket.io pour l'envoi de notifications en temps réel
  • ElasticSearch pour la recherche

Frontend

Le frontend est propulsé par les outils suivants:

  • Angular comme librairie frontend principale
  • JQuery pour l'utilisation de certaines librairies secondaires
  • Bootstrap comme framework CSS
  • Socket.io pour la réception de notifications en temps réel

Hébergement

La plateforme est déployée sur les infrastructures d'Heroku tandis que la base de données est hébergée sur MongoLab.
